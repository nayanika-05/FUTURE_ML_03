# 📄 AI Resume Screening & Candidate Ranking System

## Overview

An NLP-powered Resume Screening System that automatically analyzes resumes, extracts skills, ranks candidates, identifies skill gaps, and provides hiring recommendations.

## Features

* Resume Parsing
* Skill Extraction
* Candidate Ranking
* Skill Gap Analysis
* Hiring Recommendations
* Interactive Streamlit Dashboard

## Technologies Used

* Python
* Pandas
* Scikit-learn
* NLP (TF-IDF)
* Streamlit

## Results

The system evaluates resumes against job requirements and helps recruiters shortlist suitable candidates efficiently.
